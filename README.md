# Дигитални сат са алармом

Пројекат за курс *[назив предмета]* — имплементација дигиталног сата са алармом на микроконтролеру **AT89C51RC2** (Intel 8051 архитектура).

## Опис пројекта

Систем приказује тренутно време и време аларма на LCD дисплеју (16x2, HD44780 компатибилан), у формату `HH:MM:SS`. Време и аларм се могу подешавати на два независна начина:

- **преко тастера** (4 тастера: SET, INC, DEC, CONFIRM), кроз секвенцијални мени за подешавање сати/минута/секунди;
- **преко серијске (UART) комуникације**, слањем текстуалних команди са рачунара (нпр. преко Termite-a).

Када се тренутно време поклопи са временом аларма, на дисплеју се исписује порука обавештења.

## Хардвер

| Компонента | Пин / Порт |
|---|---|
| LCD подаци (D4-D7) | P1.0 - P1.3 |
| LCD RS | P3.3 |
| LCD EN | P3.2 |
| LCD Backlight | P1.4 |
| Тастер SET | P0.0 |
| Тастер INC | P0.1 |
| Тастер DEC | P0.2 |
| Тастер CONFIRM | P0.3 |
| UART TX | P3.1 |
| UART RX | P3.0 |

Кристал: **11.0592 MHz**

> Порт P0 je open-drain и захтева спољашње pull-up отпорнике (10kΩ) на свим коришћеним пиновима.

## Модули

**`clock`** — чува тренутно време и време аларма, инкрементује секунде преко Timer2 прекида (сваких 50ms, auto-reload режим), нуди `getTime()` / `setTime()` / `getAlarmTime()` / `setAlarm()` / `checkAlarm()` функције.

**`display`** — низак ниво управљања LCD-ом преко 4-битног интерфejsa (init, писање карактера/стрингова, позиционирање курсора).

**`command`** — прима UART карактере преко прекида, пуни бафер и препознаје завршену команду (затворена заграда), позива парсер.

**`buttons`** — машина стања (state machine) за секвенцијално подешавање: сати → минути → секунде времена, затим сати → минути → секунде аларма.

## UART протокол

Комуникациони параметри: **19200 baud, 8N1**.

Команде се шаљу као текст, затворене у заграде:

| Команда | Опис |
|---|---|
| `(SETTIME:HH:MM:SS)` | Поставља тренутно време |
| `(SETALARM:HH:MM:SS)` | Поставља време аларма |
| `(STATUS)` | Приказује тренутно време и аларм на дисплеју |

Пример: `(SETTIME:14:30:00)`

Формат мора бити тачан (двоцифрени бројеви, тачно распоређени `:` и `)`), у супротном се исписује порука о грешци на дисплеју.

## Подешавање преко тастера

1. **SET** — улазак у режим подешавања (учитава тренутно време за едитовање)
2. **INC / DEC** — повећава/смањује тренутно поље (сат/минут/секунда)
3. **CONFIRM** — потврђује поље и прелази на следеће

Редослед: сати времена → минути времена → секунде времена → сати аларма → минути аларма → секунде аларма → повратак у нормалан режим.

## Позната ограничења / могућа побољшања

- Нема заштите од истовременог подешавања преко тастера и UART-а
- Аларм се приказује само привремено на дисплеју — нема звучне (buzzer) индикације
- Формат UART команди захтева тачно двоцифрене бројеве (нпр. `08`, не `8`)
